# CSC311 ML Art Competition

Predict which of three paintings a survey respondent was looking at:
- *The Persistence of Memory* (Dalí)
- *The Starry Night* (van Gogh)
- *The Water Lily Pond* (Monet)

The final model is a **soft-voting ensemble of two GradientBoosting classifiers** — one on structured features, one on TF-IDF text features. All weights are hardcoded inline in `pred.py` (no sklearn required at inference time).

---

## Inference

| File | Purpose |
|---|---|
| `pred.py` | **Submission file.** Pure-numpy inference script. Call `predict_all(csv_path)` → list of painting names. No sklearn/pytorch required. |
| `_gen_pred.py` | Code generator — reads `model_params.npz` and writes all weights as Python literals into `pred.py`. Run after every retrain. |

---

## Training Pipeline

| File | Purpose |
|---|---|
| `train_and_save.py` | Trains the final GradientBoosting ensemble on a **70% training split** and validates on 15% val + 15% test. Saves parameters to `model_params.npz`. |
| `model_params.npz` | Compressed numpy file containing all tree arrays, TF-IDF vocab/IDF, and normalisation stats. Generated by `train_and_save.py`. |

### Rebuild workflow
```
python train_and_save.py   # train → saves model_params.npz
python _gen_pred.py        # bakes weights into pred.py
python test.py             # verify test-set accuracy
```

---

## Evaluation

| File | Purpose |
|---|---|
| `test.py` | Runs `pred.py` on the **held-out 15% test split** (same seed as training). Reports unbiased generalisation accuracy. |
| `evaluate.py` | More detailed evaluation: per-class accuracy, confusion matrix, misclassification report on the test split. |

## Model Exploration & Tuning

These scripts were used to explore model architectures and tune hyperparameters. They are **not part of the training pipeline** but are kept for reference and reproducibility.

| File/Folder | Purpose |
|---|---|
| `ensemble/ensemble.py` | Grid search over all pairs of (structured model × text model) to find the best soft-voting combination. Identified GradientBoosting + GradientBoosting as the winner. |
| `hp_sweep.py` | Hyperparameter sweep for GradientBoosting: tests different `n_estimators`, `learning_rate`, and `max_depth` values. Used to select final hyperparameters (200, 0.1, 3). |
| `decision-trees/` | Decision tree baseline experiments with various depth/criterion settings. |
| `logistic-regression/` | Logistic regression experiments with regularisation tuning. |
| `mlp/` | MLP experiments with different hidden layer sizes and learning rates. |
| `data_description.py` | EDA script: prints data summary and plots per-class histograms for all numerical features. |
